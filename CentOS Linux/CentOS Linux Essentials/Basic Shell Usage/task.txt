BASIC SHELL USAGE - TASK
=========================

TASKS:
------

1. Open terminal of your choosing.

2. Navigate to home folder of your user.

3. Create folder homework in current directory.

4. Create file file1 in homework folder.

5. Write your name and surname into file1 file.

6. View the contents of file1 in the console.

7. Navigate to the root of your file system.

8. Create file file2 in homework folder without changing current directory
   with the same timestamp as file1.

9. Find files (without directories) in your user's home directory that were
   modified less than 1 hour ago.

10. Review your task.

11. Remove homework folder.


SELF-REVIEW:
------------

- Command ls -la /<your_user_home_directory>/homework returns two files
  with the same timestamp.

- Command cat /<your_user_home_directory>/homework/file1 returns your
  name and surname.

- Your find command returns file1 and file2.

- After removing homework folder command ls -la /<your_user_home_directory>/
  does not contain homework folder.
