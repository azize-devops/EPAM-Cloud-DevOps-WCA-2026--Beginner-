MEMORY DIAGNOSTICS - TASK
=========================

TASKS:
------

1. Open terminal of your choosing.

2. Using tools of your choosing inspect RAM usage in your system (free).

3. Calculate total, free and cached RAM.

4. Find the process name that consumes the most RAM.


SELF-REVIEW:
------------

- You are able to display RAM usage using free command.

- You can identify total, free and cached memory values.

- You can find which process uses the most RAM.
