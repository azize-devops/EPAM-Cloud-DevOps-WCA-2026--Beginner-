# MANAGE DATA STRUCTURED FILE FORMATS WITH POWERSHELL - TASK

GOALS:
------

- Learn to export data to structured file formats
- Work with CSV, XML, JSON, and YAML files
- Import structured data back into PowerShell objects


TASKS:
------

1. Write a script to get list of process (ex. All chrome process) and put
   Name, Id, StartTime, PagedMemorySize in CSV/XML/JSON/YAML files
   where possible.

2. Write a script to create hash-tables based on data that stored in files
   from task 1.


SELF-REVIEW:
------------

- Task 1: Process data exported to CSV file
- Task 1: Process data exported to XML file
- Task 1: Process data exported to JSON file
- Task 1: Process data exported to YAML file (if possible)
- Task 2: CSV file imported and converted to hash table
- Task 2: XML file imported and converted to hash table
- Task 2: JSON file imported and converted to hash table
