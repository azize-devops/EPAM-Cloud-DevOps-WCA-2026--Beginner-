OUTPUT AND FORMAT VIEW - TASK
=============================

GOALS:
------

- Learn PowerShell output formatting
- Use Format-Table, Format-List cmdlets
- Implement verbose output with -Verbose parameter


TASKS:
------

1. Write a script to list directory that contain files.
   Output is the same with Get-ChildItem cmdlet.
   Print file size in megabytes (Mb).

2. Write a script to sort a list of elements using the bubble sort algorithm.
   Make output in console for each sort steps and could be controlled
   by Verbose parameter (user can choose if he would like to see output).


SELF-REVIEW:
------------

- Task 1: Directory listing matches Get-ChildItem format
- Task 1: File sizes displayed in megabytes
- Task 2: Bubble sort algorithm implemented correctly
- Task 2: Verbose output shows each sorting step
- Task 2: -Verbose switch controls output visibility
