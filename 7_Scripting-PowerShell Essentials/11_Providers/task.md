# Providers - Task

## Goals

- Understand PowerShell providers concept
- Work with FileSystem provider
- Read from and write to files using providers

## Tasks

1. Write a script to get 10 longest words chart and put them in separate file.

2. Write a script to sort words in alphabet order from specific file and put them into 26 text files named `A.txt`, `B.txt`, and so on up to `Z.txt`.

## Self-Review

- **Task 1:** Words extracted and sorted by length
- **Task 1:** Top 10 longest words saved to output file
- **Task 2:** Words sorted alphabetically
- **Task 2:** 26 files created (`A.txt` through `Z.txt`)
- **Task 2:** Each file contains words starting with that letter
