# VIRTUALIZATION INTRODUCTION - TASKS

1. Get acquainted with the structure of the user manual VirtualBox
   https://www.virtualbox.org/manual/

2. From the official VirtualBox site https://www.virtualbox.org/ download the latest stable version of VirtualBox according to the host operating system (OS) installed on the student's workplace. For Windows, the file may be called, for example, VirtualBox-6.1.34-150636-Win.exe. Install VirtualBox.

3. Download the latest stable version of Ubuntu Server or Ubuntu Desktop from the official site https://ubuntu.com/download.

4. Create VM1 and install Ubuntu using the instructions [manual VirtualBox, chapter 1.8]. Set machine name as "VM1"

5. Get acquainted with the possibilities of VM1 control - start, stop, reboot, save state, use Host key and keyboard shortcuts, mouse capture, etc. [manual VirtualBox, ch.1.9].

6. Clone an existing VM1 by creating a VM2 [manual VirtualBox, ch.1.14].
