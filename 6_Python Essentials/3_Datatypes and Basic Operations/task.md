# Datatypes and Basic Operations - Task

## Goals

- Learn to create a simple function in Python
- Understand basic arithmetic operations

## Tasks

1. Open Terminal.

2. Run `python3` (You can use virtual environment from previous task).

3. Execute:
   ```python
   def sum(a, b):
   ```

4. Execute (don't forget tabulation/indentation before print):
   ```python
       print(a + b)
   ```

5. Press Return/Enter (pass empty string) to complete function definition.

6. Execute:
   ```python
   sum(2, 4)
   ```

## Self-Review

- As the result, you should get `6` on the screen
