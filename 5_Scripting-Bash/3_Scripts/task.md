SCRIPTS - TASK
==============

GOALS:
------

- Write scripts
- Set proper script permissions
- Execute scripts


TASKS:
------

1. Open a terminal of your choosing.

2. In your user's home directory create a file named script.sh

3. Make the file executable and add a proper #! (shebang).

4. Write a script that does the following:
   - Writes "Creating folder" to console
   - Creates a folder called Name_Surname
   - Lists folders in current directory
   - Writes "Creating files" to console
   - Creates two files called Name and Surname
   - Lists contents of Name_Surname folder

5. Execute your script.


SELF-REVIEW:
------------

- You can execute the script.

- Script outputs the following into console:

  Creating folder
  Name_Surname script.sh
  Creating files
  Name Surname
