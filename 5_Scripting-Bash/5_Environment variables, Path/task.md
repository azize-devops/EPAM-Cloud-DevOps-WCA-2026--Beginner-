ENVIRONMENT VARIABLES, PATH - TASK
==================================

GOALS:
------

- Setting and unsetting environment variables
- Work with PATH


TASKS:
------

1. Create homework folder in your user's home folder.

2. Create SURNAME environment variable equal to your surname.

3. Verify that SURNAME variable is present in the environment using env command.

4. Create a script in homework folder that prints the value of SURNAME variable.

5. Add homework folder to PATH.

6. Verify that you are able to run your script by typing only its name into
   the terminal.


SELF-REVIEW:
------------

- my_script.sh command gives the following output:

  Surname
