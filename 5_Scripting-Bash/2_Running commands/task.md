# Running Commands - Task

## Goals

- Learn how to run simple commands

## Tasks

1. Open a terminal of your choosing.

2. Go to your user's home folder using `cd` command.

3. Create a folder named `homework` using `mkdir` command.

4. Create a file named `file1` in `homework` folder using `touch` command or any text editor.

5. Copy `file1` file to `homework` folder under `file2` name using `cp` command.

6. Edit `file2` using a text editor of your choosing, add `echo` command that will print hello world.

7. Make `file2` file executable.

8. Execute `file2` script.

## Self-Review

- `ls ~/homework/` command returns:

```
file1 file2
```

- script execution returns:

```
hello world
```
