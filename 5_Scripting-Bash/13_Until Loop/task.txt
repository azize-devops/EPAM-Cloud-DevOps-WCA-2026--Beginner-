UNTIL LOOP - TASK
=================

GOALS:
------

- Learn to use until loop


TASKS:
------

1. Create a script that does the following:
   - Reads a filename from user input
   - Combines specified file with itself until it reaches a size
     greater than 1024 KB

2. Create a file using the following command:
   head -c 4KB /dev/urandom > file.txt

3. Execute your script passing the file you've just created.


SELF-REVIEW:
------------

- Script returns the following output:

  Filesize: 8
  Filesize: 16
  Filesize: 32
  Filesize: 64
  Filesize: 128
  Filesize: 256
  Filesize: 504
  Filesize: 1004
  Filesize: 2004
