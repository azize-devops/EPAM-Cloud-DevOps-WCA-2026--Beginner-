# If Statement - Task

## Goals

- Practice if statement

## Tasks

1. Create a script called `odd_even.sh` that does the following:
   - Takes one argument
   - If the number of letters in the argument is odd, outputs the message "Odd"
   - Else if the number of letters is even, outputs "Even"

2. Run the script first with `"odd"` argument, then with `"even"` argument.

3. Create a script called `my_script.sh` that does the following:
   - Takes any number of arguments
   - If there are less than 2 arguments, output the values of these arguments
   - If there are more than 2 but less than 4 arguments, output only the last argument
   - Else output "Invalid number of arguments" message

4. Run `my_script.sh` with the following arguments:
   - `hello`
   - `hello world`
   - `pie is lie`
   - `keep calm and procrastinate`

## Self-Review

- `odd_even.sh` script outputs:
  - First run: `Odd`
  - Second run: `Even`

- `my_script.sh` script outputs:
  - `hello`
  - `Invalid number of arguments`
  - `lie`
  - `Invalid number of arguments`
