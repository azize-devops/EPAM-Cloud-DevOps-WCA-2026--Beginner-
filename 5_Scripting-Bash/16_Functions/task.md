# Functions - Task

## Goals

- Learn to use functions
- Learn to use nested functions

## Tasks

1. Create a script that does the following:
   - Has a function that multiplies the argument passed to it by itself
   - Has a second function that passes each argument passed to the script to the first function, increases the result by 1, and outputs to console

2. Run your script with the following arguments: `5 6 1 3 9`

## Self-Review

- Script returns the following output:

```
26
37
2
10
82
```
