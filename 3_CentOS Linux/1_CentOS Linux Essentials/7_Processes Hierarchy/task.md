# PROCESSES HIERARCHY - TASK

## TASKS

1. Open terminal of your choosing.

2. Create a script called script.sh with the following content:

   ```bash
   #!/bin/bash

   sleep 5m
   ```

3. Make the script executable and run it in background using command: `./script.sh &`

4. Verify that your script is running in the background using `jobs` command.

5. Inspect running processes in your system using `top` and `ps` commands.

6. Find the PID of your script.

7. Find command and user information about your script.

## SELF-REVIEW

- The script is running.
- You are able to determine script's PID, user and command information.
