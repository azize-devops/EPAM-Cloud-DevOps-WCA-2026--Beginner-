# CENTOS INSTALLATION AND BASIC CONFIGURATION - TASK

TASKS:
------

1. Install latest CentOS distribution on hypervisor of your choosing.

2. Configure user "Name_Surname" during installation
   (substitute Name and Surname with your credentials).

3. Configure network during installation.

4. Configure custom sizes for "/home", "/opt", "/var" directories during installation.


SELF-REVIEW:
------------

- You are able to login into VM using user "Name_Surname".

- You are able to access internet from your VM.

- Command "df -h" in terminal reflects the sizes you set for
  "/home", "/opt", "/var" directories.
