# CPU DIAGNOSTICS - TASK

TASKS:
------

1. Open terminal of your choosing.

2. Using tools of your choosing inspect CPU usage in your system (top, htop, uptime).

3. Find the number of CPU cores in your system.

4. Find current CPU load.

5. Find the process name that utilizes the most CPU.


SELF-REVIEW:
------------

- You are able to display CPU usage using top, htop, and uptime commands.

- You can identify the number of CPU cores in your system.

- You can determine the current CPU load average.

- You can find which process uses the most CPU.
